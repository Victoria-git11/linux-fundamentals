# Linux Fundamentals – Practice Repository

This repository contains practical exercises and notes based on a Linux Fundamentals course.
The focus is on command-line usage, file permissions, processes, and basic shell scripting.

The purpose of this repository is to demonstrate hands-on practice with Linux tools
commonly used in entry-level operations roles.

## Contents
- `commands.md` – frequently used Linux commands
- `scripts/` – basic shell scripts
- `exercises/` – practice tasks and solutions
